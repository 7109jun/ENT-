## Overview

Variable Data는 프로젝트 내에서 사용되는 모든 변수의 상태와 속성을 기술합니다.

프로젝트에서 정의된 각 변수의 이름, 타입, 초기값, 표시 여부, 지역 변수 여부 등을 포함합니다.

## Official Structure

| Field        | Type                                     | Official Description           |
| ------------ | ---------------------------------------- | ------------------------------ |
| id           | Variable ID                              | 변수 ID                          |
| variableType | variable \| list \| timer \| answer \| slide | 변수형                            |
| name         | string                                   | 변수명                            |
| value        | string                                   | 변수 값                           |
| minValue     | number                                   | 최솟값                            |
| maxValue     | number                                   | 최댓값                            |
| visible      | boolean                                  | 캔버스 표시 여부                      |
| x            | number                                   | 캔버스 위치 X 좌표                    |
| y            | number                                   | 캔버스 위치 Y 좌표                    |
| width        | number                                   | 표시 영역 너비                       |
| height       | number                                   | 표시 영역 높이                       |
| isCloud      | boolean                                  | 공유 변수 여부                       |
| object       | Object ID                                | 지역 변수일 경우 참조하는 Object ID       |
| array        | Array Data                               | variableType이 list인 경우 저장되는 배열 |
